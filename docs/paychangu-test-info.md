# PayChangu Test Information

## Test Cards
For card payments during testing, use these test cards:
- OTP for 3DS: `1234`

## Mobile Money Test Numbers
Based on PayChangu documentation, use these formats for testing:
- Airtel Money: `+265990xxxx00` (use `0990000000` format in payment forms)
- TNM Mpamba: Similar format (check PayChangu dashboard for specific test numbers)

## Test Mode Setup
1. Use test API keys from your PayChangu Dashboard
2. Ensure test mode is enabled in your PayChangu account
3. Use test amounts above 50 MWK (minimum requirement)

## Common Test Numbers to Try
- `0990000000` (Airtel Money test format)
- `0880000000` (TNM Mpamba test format)
- `+265990000000` (full international format)

## Notes
- Real money will NOT be charged in test mode
- These numbers simulate successful payments
- If you have specific test numbers from your PayChangu dashboard, use those instead
